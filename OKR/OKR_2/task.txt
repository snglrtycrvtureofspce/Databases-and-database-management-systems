1. Создате и заполните таблицу базу данных "Физические свойства металлов (при 20 градусов C)"
Таблица:
Металл; плотность в кг/м^3; температура плавления в градусов C; удельная теплопроводность в Вт/(м * К)
Данные:
алюминий; 2700; 659; 218
железо; 7870; 1539; 73
золото; 19300; 1063; 312
медь; 8940; 1083; 394
никель; 8960; 1455; 75
литий; 533; 180; 71
Решение:

CREATE TABLE Metals (
    Металл NVARCHAR(50),
    Плотность DECIMAL(10,2),
    ТемператураПлавления INT,
    УдельнаяТеплопроводность DECIMAL(10,2)
);

INSERT INTO Metals (Металл, Плотность, ТемператураПлавления, УдельнаяТеплопроводность)
VALUES

    ('Железо', 7870, 1539, 73),
    ('Золото', 19300, 1063, 312),
    ('Медь', 8940, 1083, 394),
    ('Никель', 8960, 1455, 75),
    ('Литий', 533, 180, 71);

2. Создайте табличную форму для отображения иформации из таблицы базы данных.
3. На форме расположите кнопки для добавления и сохранения в базу данных информации о новом металле.
4. На форме расположите кнопку для удаления информации из базы данных о выбранном металле.
5. Создайте меню. За пунктами меню закрепите вызов табличных форм, на котором отразите в объекте DataGridView результаты следующих запросов:
а) Определите металлы, у которых температура плавления больше плотности. В таблице результатов запроса указать металл, температура плавления, плотность.
б) Определить металл с минимальной удельной теплопроводностью. В таблицу результатов запроса указать название металла, удельную теплопроводность.
в) Определить металлы, у которых удельная теплопроводность в заданном диапазоне. В таблицу результатов запроса указать поля название металла, удельная теплопроводность.
6. На главной форме создайте кнопку "Отчёт". За кнопкой закрепите вызов отчёта. Поля отчёта: металл, плотность, удельная теплопроводность (в порядке убывания). В отчёте поместите те записи, у которых удельная теплопроводность больше 75.
7. Подведите итоги: найти среднюю плотность всех металлов; найти металл с наибольшей температурой плавления.
